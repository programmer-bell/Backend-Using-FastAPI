# Backend-Using-FastAPI
Here , i create a  project's(Library Management System ) backend logics using FastAPI. In future i want to add react and more functionality here.
# Firstly, you need a virtual environment:
 => python -m venv venv
# Now active your venv
=> venv/Scripts/activate
# Also, need postgres for database creation , open postgres sql , create the database  name 'library', you can use othres databasr well.If, another database, you need to change the .env file.
# After that, install all requirements
=> pip install -r requirements.txt
# Now run the project
=>python run.py
